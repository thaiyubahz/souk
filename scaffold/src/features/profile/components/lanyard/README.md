# Lanyard assets — REPLACE BEFORE RUNNING

This folder ships with **placeholder** `card.glb` and `lanyard.png` files so TypeScript compiles. Before you run the app, you MUST replace both with the real assets from React Bits:

- **Source**: https://github.com/DavidHDev/react-bits → `src/assets/lanyard/card.glb` and `lanyard.png`
- Drop them into this directory, overwriting the placeholders. Keep the filenames exactly the same.

You can edit the card's baked-in texture in https://modelviewer.dev/editor/ (open the .glb, replace the texture, download the new .glb).
